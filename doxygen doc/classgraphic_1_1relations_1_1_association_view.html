Detailed Description

The LineView class represent a collection of lines making a link between two GraphicComponent.

When it creates, the LineView have one single line between the two GraphicComponent. By clicking on the line, the user can personnalize the LineView by adding new grips. When drawing, the LineView will draw a segment between each grips. Grips are movable and a LineView have two special grips; MagneticGrip. These grips are associated with a GraphicComponent and can't be placed elsewhere.

A RelationView have an associated UML component.

An AssociationView is associated with an association UML component.
