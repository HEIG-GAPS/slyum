Detailed Description

Each datafield has a specific type.

This type can be a "Primitive type" (int, double, char, etc) a user defined type (an object created by the user) or an API type. This class represent an API interface. It implements Implementable. Implementable represent an interface regrouping the API interfaces and the user defined intefaces
