Detailed Description

A TextBox is a graphic component from Slyum containing a String.

The particularity of a TextBox is it can be moved with mouse and its String can be edited by double-click on it.

A TextBoxMethod is a TextBox displaying a Method (UML). When editing the text, this TextBox parse the String to change it into a Method. It listening Method changes for auto-update itself.
