Detailed Description

A TextBox is a graphic component from Slyum containing a String.

The particularity of a TextBox is it text can be changed by double-clinking on it.

David Miserez

void graphic.textbox.TextBox.editing ( )

This method change the mode of the TextBox.

Calls this method for turn the TextBox in edit mode, allow users to change the String with a JTextField. For stop editing, call manually the method stopEditing() or the user can accept or reject edition of the String by pressing enter or esc key.
