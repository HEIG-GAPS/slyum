Create a new composition between the entity source and target.

Two news roles will be created containing this association and an entity (source or target). Entities will add this new role corresponding to this association. The parameter directed define if the association is directed or not. An association directed means, in UML, that an association can be read only in one direction (represented by an arrow).

Parameters:
