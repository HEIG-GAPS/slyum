Detailed Description

This class is the main container for all diagrams components view (classComponent, interfaceComponent, relation, ...).

It performs class diagram, movables and resizables events. Each graphic component have a reference on the graphic view.

Author:
David Miserez
Version:
1.0 - 25.07.2011

graphic.GraphicView.GraphicView ( ClassDiagram  classDiagram)

Create a new graphic view representing the class diagram given.

The new graphic view is empty when created. If classDiagram given is not empty, you must manually add existing component.

Parameters:
