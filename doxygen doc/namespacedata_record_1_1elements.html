Detailed Description

Each datafield has a specific type.

This class represent a parameter.

This class represent a comment made in a source code file A code can be of one single line or multiline.

This class represent a class in an object-oriented language.

This type can be a "Primitive type" (int, double, char, etc) a user defined type (an object created by the user) or an API type. This class represent a data field. A data field is usually with the following format {AccessModifier} {static} {final} type name {= value} example: final String a = "un attribute" private int entier
