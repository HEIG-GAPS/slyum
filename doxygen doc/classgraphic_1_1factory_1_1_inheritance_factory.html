Detailed Description

InheritanceFactory allows to create a new inheritance view associated with a new association UML.

Give this factory at the graphic view using the method initNewComponent() for initialize a new factory. Next, graphic view will use the factory to allow creation of a new component, according to the specificity of the factory.
