static int utility.Utility.getColorGrayLevel ( Color  color) [static]

Compute the gray level of a color.

The gray level is calculated by adding the hightest RGB component with the lowest and divided by 2. The gray level is a float value between 0.0 and 1.0f. This method return a color with RGB equals 100 if the gray level is less than 0.5f, 200 otherwise. It permit to find a gray color who are visible on the given color (except if the color is a gray near of 127 RGB).

color
Returns:
