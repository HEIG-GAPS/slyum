Detailed Description

The relationGrip is a grip who customize a LineView.

The LineView uses RelationGrip for draw segments between each RelationGrip.

The MagneticGrip is a RelationGrip that is associated with a GraphicComponent. The MagneticGrip can be moved but it compute its location by calling the computeAnchorLocation() from the GraphicComponent that is assigned to the grip each time the setAnchor() si called. A LineView have two MagneticGrip for each extremity.
