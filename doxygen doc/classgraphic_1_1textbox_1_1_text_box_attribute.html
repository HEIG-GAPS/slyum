Detailed Description

A TextBox is a graphic component from Slyum containing a String.

The particularity of a TextBox is it can be moved with mouse and its String can be edited by double-click on it.

A TextBoxAttribute is a TextBox displaying an Attribute (UML). When editing the text, this TextBox parse the String to change it into an Attribute. It listening Attribute changes for auto-update itself.
