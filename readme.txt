Installation :

No installation required.

Slyum works with Java, see Java download page for more informations (Slyum needs jre 7).

	http://www.oracle.com/technetwork/java/javase/downloads/index.html.

- Windows or Linux

	Double-click on Slyum.jar

	!!!!! Warning : OpenJDK is strongly discouraged on Linux system !!!!!


- Mac Os X
	
	Double-click on SlyumMacOsX.jar

Dependency required : documentation/User manual.pdf
